## Follow instructions
1. Clone this repo:

       git clone https://github.com/sangwansangwan/Golang-99design-boilerplate.git

2. Run tidy command: 

       go mod tidy

3. Initialise gqlgen config and generate models

       go run github.com/99designs/gqlgen 

4. Run server 

       go run server.go
